\doxysubsection{Detailed Description}
A structure containing type information and operations for a component. 

This structure holds metadata and function pointers for managing the lifecycle of a component type. It includes functions for construction, destruction, copy construction, copy assignment, move construction, move assignment, and combined move and destruct operations.
