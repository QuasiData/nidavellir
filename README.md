<!-- GETTING STARTED -->
## Getting Started

Setup

### Prerequisites

* cmake https://cmake.org/

### Installation

1. Clone the repo
   ```sh
   git clone --recursive https://github.com/QuasiData/nidavellir.git
   ```
2. Move to the repo
    ```sh
    cd nidavellir
    ```

3. Build and run tests
    ```sh
    cmake --workflow --preset=x64-{platform}-{compiler}-{config}
    ```

4. Example
    ```sh
    cmake --workflow --preset=x64-linux-gcc-release
    ```
